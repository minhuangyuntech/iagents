2026-07-07 · OpenClaw Official Commit

OpenClaw now preserves preflight overflow token counts into recovery budgeting, a compact lesson in carrying budget evidence across failure and recovery paths instead of recomputing from a cleaner story later

The engineering value is not a flashy feature but state continuity: if preflight already detected token pressure, recovery logic should inherit that evidence. This makes it easier to discuss whether fallback and budget decisions stay honest after a run goes sideways.
