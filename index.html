OpenClaw

Your own personal AI assistant. Any OS. Any Platform. The lobster way.

官方網站 GitHub Repo 官方文件

中文理解重點

  • 它強調的是「你自己的 AI 助手」，而不是只有單一 CLI 工具。
  • 支援多種訊息通道與平台整合，適合從真實日常溝通介面切入。
  • 官方建議從 onboard / getting started 開始，逐步完成安裝與通道設定。
  • 若要做中文教學，可以整理成「安裝」、「通道」、「使用情境」、「進階擴充」四大模組。

Hermes Agent

The self-improving AI agent built by Nous Research.

官方網站 GitHub Repo 官方文件

中文理解重點

  • Hermes Agent 的核心特色是 memory、skills、tools、delegate 與跨平台 gateway。
  • 它不只是聊天工具，而是具備可持續學習與工作流能力的 agent 系統。
  • 若用於中文教材，可優先整理 CLI、工具呼叫、技能系統、背景任務與平台整合。

建議學習路線

  1. 先閱讀專案官網與 Getting Started，建立整體概念。
  2. 再看 GitHub Repo 與 release notes，理解專案更新節奏。
  3. 接著透過 YouTube 搜尋入口找安裝與實作影片。
  4. 親自部署一個最小可運作版本，實測通道、工具或 CLI。
  5. 最後再回到中文筆記與比較整理，建立自己的教學或研究工作流。
