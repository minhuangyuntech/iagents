Hermes Agent 中文詳頁

Hermes Agent 是 Nous Research 打造的 self-improving AI agent。它的重點不只是聊天，而是 memory、skills、tools、delegate 與跨平台 gateway 所構成的完整代理系統。

先讀這三個

官方網站 官方文件 GitHub Repo

核心理解重點

  • 工具呼叫能力讓 agent 能夠真正執行檔案、終端機、瀏覽器等操作。
  • 記憶系統讓長期互動與個人化工作流成為可能。
  • 技能系統可把複雜流程封裝成可重複使用的操作知識。
  • Delegate / background tasks 代表它已經是工作流型 agent，而非單輪對話機器人。

建議學習順序

  1. 先看 Getting Started，理解最基本使用方式。
  2. 再看 docs 中與 tools、memory、skills 相關說明。
  3. 閱讀 release notes，理解近期功能演進方向。
  4. 最後再研究 gateway、MCP、delegate 等進階能力。
