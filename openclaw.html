OpenClaw 中文詳頁

OpenClaw 是以「你自己的個人 AI 助手」為核心定位的專案。它強調跨平台整合、真實日常通訊介面的使用情境，以及從安裝、通道到互動體驗的一體化設計。

快速入口

官方網站 官方文件 GitHub Repo

你可以怎麼理解 OpenClaw

  • 不是單純 CLI，而是面向真實使用場景的個人 AI 助手。
  • 強調 Any OS、Any Platform，適合從跨平台互動角度理解。
  • 很適合示範通道整合、訊息介面與個人化助理的產品設計。
  • 也適合作為課堂上「AI 助手產品化」的案例。

建議學習順序

  1. 先看官方網站，理解產品定位。
  2. 閱讀 Getting Started，建立最小安裝概念。
  3. 查看 Showcase 與 FAQ，理解應用情境。
  4. 最後再看 Docker / DeepWiki / GitHub 結構。
